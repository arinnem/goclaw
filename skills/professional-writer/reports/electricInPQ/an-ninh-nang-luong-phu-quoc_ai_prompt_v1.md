# AI Presentation Prompt

## Instruction

Create a professional presentation with the following specifications. Follow the content, structure, and style guidelines exactly as described below. Each slide must contain the provided content — do not add, remove, or rephrase the key messages. This is a strategic proposal for executive leadership — the tone must be authoritative, data-driven, and urgent.

## Specifications

- **Title:** Đảm bảo an ninh năng lượng cho danh mục đầu tư Sun Group tại Phú Quốc
- **Total slides:** 15
- **Duration:** 16 minutes
- **Language:** Tiếng Việt
- **Style/Vibe:** Professional formal, executive briefing, data-driven with emotional pain points
- **Color scheme:** Dark professional — primary dark navy (#1a1a2e), accent red (#e74c3c) for pain/risk, accent green (#27ae60) for solutions. White text.
- **Font:** Sans-serif modern (Montserrat for headings, Inter or Noto Sans for body)

## Design Guidelines

- Clean, dark layout with ample whitespace — executive feel
- One key idea per slide — maximum 5 bullets
- Use red accent for risk/problem slides, green accent for solution slides
- Tables with thin borders, no heavy gridlines
- Charts/diagrams where indicated — simple, high-contrast
- No decorative images — icons and diagrams only
- Consistent heading style: left-aligned, large
- Vietnamese text — no unnecessary English
